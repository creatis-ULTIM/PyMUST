PyMUST tutorial
--------------

The tutorials correspond to an undergraduate course on ultrasound signal processing given to 2nd/3rd year biomedical engineering students at Universitat Pompeu Fabra (Barcelona, Spain). It consists of python notebooks which they need to complete, illustrating different concepts of the process of ultrasound image generation.
